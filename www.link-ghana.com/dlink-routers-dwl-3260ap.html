D-Link DWL-3260AP/BEU - Wireless 108Mbps 11g Ceiling Mount Access Point with PoE
  • Key Features
    • DWL-3260AP D-Link AirPremier - a high-performance wireless access point for enterprise networks, designed for indoor installation. It can perform the functions of the base station to connect to the wireless network devices operating on the 802.11b and 802.11g standards. Due to the high transmission speed of up to 108 Mbit / s (Turbo-mode), the advanced security features, support the QoS, PoE Access Point provides a solution to "business class" for the deployment of high-performance and secure wireless networks
    • The connection speed up to 108Mbps / sec
      DWL-3260AP supports the 802.11g standard, in which the transfer rate up to 54Mbps / s, providing the connection, and through the use of 108G D-Link technology, this access point is able to reach a maximum wireless signal rate - up 108 Mbit / s (Turbo-mode). At the same time, DWL-3260AP is fully compatible with 802.11b and 802.11g standards.
    • Supports Power Over Ethernet (PoE).
      To increase coverage DWL-3260AP can be placed in a remote place, for example, on the ceiling, where power outlets are not available, and providing power to these locations is difficult and expensive task. DWL-3260AP access point can be powered via the existing network cable from a PoE switch located at a distance of no more than 100 meters. With support for 802.3af PoE standard, the access point does not require a PoE injector.
    • Mounting on the ceiling.
      DWL-3260AP office looks like a rounded shape smoke detector and does not attract the attention of network intruders. Even more like a smoke detector, you can disable the LED device.
    • Advanced wireless security.
      When deploying a wireless network with one of the important issues is the problem of ensuring its security. In order to ensure complete network protection DWL-3260AP supports WPA-Enterprise standards, 802.1x and 802.11i. Other security features include MAC-address filtering, segmentation of WLAN, SSID broadcasting ban and support for Advanced Encryption Standard (AES) encryption.
    • Mode support WDS (Wireless Distribution System).
      To improve the return on investment, DWL-3260AP can be configured to operate in a "point-to-point" mode access point (AP mode), a bridge or a "point - multipointÂ» (WDS mode).
    • Increasing the flexibility and efficiency of the network.
      DWL-3260AP supports multiple SSID (Multiple SSID), that allows you to divide the access point into several independent segments based on security and performance requirements. Basically SSID can enable encryption and authentication to protect a private network, SSID and the other left open to the public.
    • Saving and Mobile Applications.
      DWL-3260AP Access Point supports multiple SSID (Multiple SSID), allowing the administrators to logically divide the access point into several virtual access points all within a single hardware platform. Rather than having two separate networks with two access points, administrators can deploy one single AP to support more than one application, such as public Internet access and internal network control to increase flexibility and keep costs down.
    • Advanced Network Management.
      Network administrators can manage all the DWL-3260AP via the configuration utility Web-based interface or via Telnet. For advanced network management, administrators can use the D-Link AP Manager or Control Module D-View SNMP to configure and manage multiple access points from a single location.
